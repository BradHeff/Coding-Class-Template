## Getting Started

To get a local copy up and running follow these simple example steps.

Software needed for this application to work
* python 3.8
* * Windows: 
* * * [Download Python](python-url)
* * Linux:
* * *  ```sudo apt install python3```

1. Clone the repo
